# JavaBean-my-own-JTable

JavaBean creado con fines académicos que extiende de JTable de Swing y le agrega funcionalidades (o facilita existentes) para: Alinear horizontal, modificar bordes de celdas, cambiar color de fondo, cambiar color de letras, pintar una fila, cargar datos, vaciar datos e imprimir datos de una fila en un messageDialog.
